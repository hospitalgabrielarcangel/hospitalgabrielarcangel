# [Hospital El Camino a Gabriel Arcángel](https://hospitalgabrielarcangel.com)

This is the official [El Camino a Gabriel Arcángel](https://hospitalgabrielarcangel.com) website.

[![El Camino a Gabriel Arcángel](https://hospitalgabrielarcangel.com/opengraph-image.jpg)](https://hospitalgabrielarcangel.com/)

## Tech Stack

- **Environment:** [Bun](https://bun.com)
- **Framework:** [Next.js](https://nextjs.org)
- **Styling:** [Tailwind CSS](https://tailwindcss.com)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com)
- **Validation:** [Zod](https://zod.dev)
- **Internationalization:** [next-intl](https://next-intl.dev)

## Running Locally

1. Clone the repository

   ```bash
   git clone https://github.com/hospitalgabrielarcangel/hospitalgabrielarcangel.git
   ```

2. Install dependencies using bun

   ```bash
   bun install
   ```

3. Copy the `.env.example` to `.env` and update the variables.

   ```bash
   cp .env.example .env.local
   ```

4. Start the development server

   ```bash
   bun run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
